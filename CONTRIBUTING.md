# Contributing

This is a small Windows utility maintained primarily for personal use, but focused bug reports and pull requests are welcome.

Before submitting a change:

1. Do not commit build outputs, local BetterNCM data, certificates, cookies, or account information.
2. Run .\scripts\test.ps1.
3. Keep the bridge protocol backward-compatible or bump its version explicitly.
4. Test taskbar click-through behavior and BetterNCM reconnect behavior when changing window or bridge code.
