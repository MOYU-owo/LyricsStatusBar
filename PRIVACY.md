# Privacy

Lyrics StatusBar does not send analytics, telemetry, cookies, lyrics, listening history, or account information to the developer or to any service operated by this project.

The application communicates only with the local BetterNCM bridge through a current-user named pipe and transient JSON files in the local BetterNCM data directory. It does not open a TCP or UDP port. Current lyric and track data are held in memory and are not added to the application settings file.

NetEase Cloud Music and BetterNCM are separate third-party applications governed by their own privacy practices.
