# Security policy

Please do not include NetEase Cloud Music cookies, account details, local paths containing personal information, or other secrets in a public issue.

For a suspected vulnerability, open a GitHub security advisory after the repository is published. For ordinary bugs, use GitHub Issues and attach only redacted diagnostics.

The application does not open a network port. Lyrics are kept in memory; the BetterNCM file bridge stores only transient current-session messages in the local BetterNCM data directory.
